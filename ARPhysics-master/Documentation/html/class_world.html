Detailed Description

The World class is the base class representing an area where simulation can take place.

Constructor for World object.

Parameters
widthThe width of the world.
heightThe height of the world.
indexingThe spatial indexing system to use for the world.
numberOfIterationsThe number of times to solve the collisions and constraints each step. The larger this number the more accurate the simulation will be but the slower it will run. The default is 5. Note that bodies can go beyond the bounds of the world making the world size arbitrary.
